# 使用 VuePress 构建个人博客

- VuePress、Valine
- 我的博客示例：[yangtao.site/vuepress/](https://www.yangtao.site/vuepress/)

## 开始

```bash
git clone https://github.com/yangtao2o/vuepress.git

cd vuepress

# 安装依赖
yarn

# 启动一个本地服务
yarn docs:dev # 或者：npm run docs:dev

# 生成静态的 HTML 文件
yarn docs:build # 或者：npm run docs:build

# 构建
yarn deploy
```
